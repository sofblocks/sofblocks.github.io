## Spawning a Shell:
Python:
```python
    python -c  'import pty; pty.spawn("/bin/sh")'
    python3 -c 'import pty; pty.spawn("/bin/sh")'
    python3 -c 'import pty; pty.spawn("/bin/bash")'
```
Echo:
```python
   echo os.system('/bin/bash') 
```
Bash and Sh:
```python
   /bin/bash -i
   /bin/sh -i
```
